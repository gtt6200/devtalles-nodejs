# Node TS multiplication app
1.-  Install deps
```
npm install 
```
2.- Run dev app
```
npm run dev
```
3.- Run dev and watch app with nodemon
```
npm run dev:watch
```
4.-  Genearted dist folder
```
npm run build
```
5.-  Start project 
###### This command build and start the project
```
npm run start
```
